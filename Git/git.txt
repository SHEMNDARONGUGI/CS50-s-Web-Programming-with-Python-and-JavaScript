1. Synchronizes code between different people
2. Test changes to code without losing original access - using branches
3. Revert back to old versions of code.

Github - site that store projects in repositories
git clone - downloading a repository to local computer

git pull - to see changes from the other branch

git commit -am "message" - tracks previously track without using git add

git log - t see what has been happening/commits done and by who

git reset --hard <commit hash>

git reset --hard origin/master

Git branches
============

git branch - to see branch

git checkout -b <new-branch name>
git checkout <branch name> to switch between existing branch
master - main branch

git merge <branch-name>

git fork - making a copy of the projects
pull request - made to the repository owner to see if they'll accept changes
